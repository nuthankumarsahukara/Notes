HTML Attributes and Parameters:

Attributes:

1.HTML tags can contain one or more attributes.
2.Attributes are always specified in the start tag.
3.Attributes consist of name/value pairs.
4.Attribute values must always be enclosed in quotes.
5.Double quotes are most commonly used, but single quotes are also allowed.
6.Attributes are special features of a tag that modify its behavior or appearance.

Parameters: Parameters are the values assigned to an attribute.

Syntax: <tagname attribute="value">Content</tagname>
Example: <body  bgcolor="blue"></body>


Body Section :-
<body> Tag: The body section contains all the contents of an HTML document, such as text, hyperlinks, images, Special Character, lists, tables, frames, forms etc. Mostly all the HTML Tags are use this Section. It's most Powerful section. It is a paired tag.

Syntax: <body>.......................</body>

Color Values

HTML colors are defined using a hexadecimal notation (HEX) for the combination of Red, Green, and Blue color values (RGB). The lowest value that can be given to one of the light sources is 0 (in HEX: 00). The highest value is 255 (in HEX: FF). HEX values are specified as 3 pairs of two-digit numbers, starting with a # sign.

HTML Colors - Hex Codes:

A hexadecimal is a 6 digit representation of a color. The first two digits(RR) represent a red value, the next two are a green value(GG), and the last are the blue value(BB).

A hexadecimal value can be taken from any graphics software like Adobe Photoshop, Advanced Paint Brush.

Hexadecimal Value - #FFFFFF RGB Color Code - R=255 G=255 B=255

16 Million Different Colors

The combination of Red, Green, and Blue values from 0 to 255, gives more than 16 million different colors (256 x 256 x 256).

Using Emojis in HTML:

Emojis look like images, or icons, but they are not. They are letters (characters) from the UTF-8 (Unicode) character set.

Emojis are special characters from the UTF-8 character set, just like letters or symbols.

Each emoji has a unique Unicode number, which can be used in HTML.

Unicode numbers are usually written as decimal or hexadecimal codes.
