Basics of Web:
Define Web ?
The Web is a collection of electronic resources (e-resources) that are accessed through the Internet.
The World Wide Web (WWW) was invented by Sir Tim Berners-Lee in 1989.

Define W3C?
W3C (World Wide Web Consortium) is an international organization that develops and maintains web standards.
It was founded by Tim Berners-Lee in 1994.

Define INTERNET?
	The Internet is a network of networks that allows people to send messages, browse websites, watch videos, download files, and access online services from anywhere in the world.The Father of the Internet is Dr. Vint Cerf.

-->Worldwide connection of computers
-->Uses TCP/IP protocols
-->Enables communication (email, chat, video calls)
-->Provides information (websites, online learning)
-->Supports services (online shopping, banking)

Ex:-Using Google, WhatsApp, YouTube, or online classes requires the Internet.

Define TCP ?
TCP (Transmission Control Protocol)
It provides reliable communication between computers in a network (LAN) and ensures proper transmission of data.

Define IP ?
IP (Internet Protocol)
It provides connectivity between networks (WAN) and is responsible for addressing and routing data packets.

Define Protocol ?
A protocol is a collection of rules followed by the sender and receiver to ensure proper communication.
(or)
It is a standard set of rules that allows electronic devices to communicate with each other over a network.

Define Email / WebMail ?
Email is a method of exchanging messages electronically between people using devices.

Webmail is a way of sending and receiving emails through a web browser.

Mail communications are classified into the following two types:
1. Online Mail Communications
2. Offline Mail Communications

1. Online Mail Communications
-->Online mail communications are popularly known as Web Mails.
-->Web mail services allow users to send, receive, and manage emails through a web browser using an Internet connection.
-->The first web mail service, Hotmail, was developed by an Indian, Mr. Sabeer Bhatia.

Example:Gmail, Yahoomail, Hotmail,Outlook ..etc,

2. Offline Mail Communications
-->Offline mail communications are popularly known as POP mails.
-->POP stands for Post Office Protocol.
-->In this method, emails are downloaded from the mail server to the user’s computer and can be read without an Internet connection.
-->These are handled using special email software applications.

Example:- MSOutlookExpress ,MozillaThunderBird, PoP3 Server.

Define SMTP?
SMTP (Simple Mail Transfer Protocol) is a standard Internet protocol used for sending emails between mail clients and mail servers, and between different mail servers (cross mail domain communication).

Define MIME?
MIME (Multipurpose Internet Mail Extensions) is an Internet standard that extends the email format to support non-text content such as images, audio, video, documents, and other file types.
It also supports Unicode and multiple character sets.

Define HTTP/HTTPs
HTTP (Hypertext Transfer Protocol) is an application-layer protocol used for transferring web pages and data between a web browser (client) and a web server on the Internet.
It is not secure because data is sent in plain text.
-->Uses port number 80
-->No Data Encryption
-->Not preferred by Google

HTTPS (Hypertext Transfer Protocol Secure) is the secure version of HTTP.
It uses SSL/TLS encryption to protect data during transmission, ensuring confidentiality, integrity, and authentication.
-->Uses port number 443
-->Data Encryption
-->Preferred by Google

Uniform Resource Identifiers (URIs)

URN --> Uniform Resource Name
	A URN is a unique name that identifies a resource independently of its location on the Internet.

URI --> Uniform Resource Identifier
	A URI is a string of characters that identifies a resource on the Internet, either by name, location, or both.

URL --> Uniform Resource Locator
	A URL is a unique address used to identify a resource on the World Wide Web.

Example:
https://www.datapro.in

Explanation:
1.https → HyperText Transfer Protocol Secure (protocol used to access the website securely)
:// → Delimiter (separates the protocol from the rest of the URL)
www → Subdomain (optional part of the domain, often used for websites)
datapro → Primary Level Domain (PLD) (main name of the website)
.in → Top Level Domain (TLD) (represents country code or type of organization, here .in = India ,.com → Commercial Domain,.org → Organization Domain,edu → Education Domain,.gov → Government Domain,.net → Network Domain,.mil → Military Domain)

WebPage:
A web page is an electronic document developed using HTML.

Types of Web Pages:
1. Static Web Pages 
2. Dynamic Web Pages

Static Web Pages
-->Written using HTML, CSS.
-->Content does not change dynamically.
-->Same content is shown to all users
-->No database is used

Dynamic Web Pages
-->Written using Java, PHP, Python, JavaScript, etc.
-->Content changes based on user interaction.
-->Connected to a database

WebSite
A collection of web pages is called a website.

Full Stack
Full Stack refers to the combination of client-side (frontend) and server-side (backend) development.
