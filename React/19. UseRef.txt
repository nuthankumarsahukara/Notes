1. What is useRef?
useRef is a hook in React used to persist a mutable reference that does not cause a re-render when updated.

Commonly used to access DOM elements or store mutable values.

useRef can store values that persist across renders without causing a re-render.

ref={...} is used to attach a ref object to a DOM element.

2. Syntax
import { useRef } from 'react';

const myRef = useRef(initialValue);

3. Use Cases
a) Accessing DOM elements

function InputFocus() {
  const inputRef = useRef(null);

  const focusInput = () => {
    inputRef.current.focus();
  };

  return (
    <>
      <input ref={inputRef} type="text" />
      <button onClick={focusInput}>Focus Input</button>
    </>
  );
}

Avoiding Re-renders on Value Changes
useRef changes do not trigger re-renders, unlike useState.

Difference: useRef vs createRef
createRef: Used in class components; creates a new ref every render.

useRef: Used in function components; persists the same ref throughout the componentâ€™s lifecycle.

When to Use useRef :-
Accessing or manipulating DOM nodes.
Storing values across renders without triggering re-renders.
Keeping track of previous props or state values.

Ex:-
import { useEffect, useRef, useState } from "react";

function App(){
  let counter=useRef(0);
  let text=useRef("");

  let[count,setCount]=useState(0);
  useEffect(()=>{
    console.log("useEffect is Calling....");
  })
  return(
    <>
      <h1>By Using useState </h1>
      <h3>Count :{count}</h3>
      <button onClick={()=>setCount(count--)}>Minus</button>
      <button onClick={()=>setCount(count++)}>Plus</button>

      <h1>By Using useRef</h1>
      <h3 ref={text}>Counter : 0</h3>
      <button onClick={()=>{
        counter.current--;
        text.current.innerText="Counter : "+counter.current;
      }}>Minus</button>
      <button onClick={()=>{
        counter.current++;
        text.current.innerText="Counter : "+counter.current;
      }}>Plus</button>
    </>
  )
}

export default App;
In Class Component :

import  { Component, createRef } from "react";

class App extends Component {

  constructor(props) {
    super(props);

    // useState equivalent
    this.state = {
      count: 0
    };

    // useRef equivalent
    this.counter = 0;
    this.text = createRef();
  }

  componentDidUpdate() {
    console.log("useEffect is Calling....");
  }

  render() {
    return (
      <>
        <h1>By Using state</h1>
        <h3>Count : {this.state.count}</h3>

        <button onClick={() =>
          this.setState({ count: this.state.count - 1 })
        }>Minus</button>

        <button onClick={() =>
          this.setState({ count: this.state.count + 1 })
        }>Plus</button>
          

        <h1>By Using ref</h1>
        <h3 ref={this.text}>Counter : 0</h3>

        <button onClick={() => {
          this.counter--;
          this.text.current.innerText = "Counter : " + this.counter;
        }}>Minus </button>

        <button onClick={() => {
          this.counter++;
          this.text.current.innerText =
            "Counter : " + this.counter;
        }}>Plus</button>
      </>
    );
  }
}

export default App;
