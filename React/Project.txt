To create React Project
	->npx create-react-app project_name
After creating the project
	->cd project_name
	->npm start
In react project file are saved in file_name.js
The flow of the react program
	->app.js ----> index.js ---->index.html
To Stop The server
	->press ctrl + c;
	->Terminate batch job (Y/N)? y

To Run the Server
	->first goto the project folder in Terminal
	->npm start


To create Vite Project
	->npx create-vite project_name
	->Select FrameWork (React JS)
	->Select Variant (JavaScript+tsx)
After creating the project
	->cd project_name
	->npm install
	->npm run dev
In vite project file are saved in file_name.jsx
The flow of the react program
	->app.js ----> main.js ---->index.html
To Stop The server
	->press ctrl + c;
	->Terminate batch job (Y/N)? y
To Run the Server
	->first goto the project folder in Terminal
	->npm run dev
