What is React Router?
React Router is a standard library for routing in React. It enables navigation between views or components without reloading the page, maintaining a Single Page Application (SPA) experience.

2. Installation
npm install react-router-dom

3. Basic Setup (BrowserRouter)

import { BrowserRouter, Routes, Route } from "react-router-dom";
import Home from "./Home";
import About from "./About";

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
      </Routes>
    </BrowserRouter>
  );
}

4. Main Components
Component	Description
<BrowserRouter>	Wraps the whole app, uses HTML5 history API
<Routes>		Parent container for <Route>s
<Route>		Defines the path and component to render
<Link>		Anchor tag alternative for navigation (no page reload)
<NavLink>	Like <Link> but with active styling
useNavigate()	Hook to programmatically navigate
useParams()	Hook to access URL parameters
useLocation()	Access current location (URL info)
useSearchParams()	Access query string parameters

5. Example with Link and useNavigate
import { Link, useNavigate } from "react-router-dom";

function Home() {
  const navigate = useNavigate();

  const handleClick = () => {
    navigate("/about");
  };

  return (
    <div>
      <h1>Home</h1>
      <Link to="/about">Go to About</Link>
      <button onClick={handleClick}>Go to About (Programmatically)</button>
    </div>
  );
}

6. Dynamic Routing
<Route path="/user/:id" element={<User />} />
Inside User.jsx:

import { useParams } from "react-router-dom";
const { id } = useParams();

7. Nested Routes
