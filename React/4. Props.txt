React Props :

Props stands for properties.
Props are used to pass data from a parent component to a child component in React.
Props are read-only and immutable, meaning they cannot be modified within the child component.
Props allow component reusability and dynamic rendering.
Props are arguments passed into React components.
Props are passed to components via HTML attributes.

1. Accessing Props in the Child Component
Props are received as function arguments in functional components or via this.props in class components.

Functional Component:

function ChildComponent(props) {
  return (
    <div>
      <h2>Name: {props.name}</h2>
      <p>Age: {props.age}</p>
    </div>
  );
}
export default ChildComponent;

Class Component:

import React, { Component } from "react";
class ChildComponent extends Component {
  render() {
    return (
      <div>
        <h2>Name: {this.props.name}</h2>
        <p>Age: {this.props.age}</p>
      </div>
    );
  }
}

export default ChildComponent;


2. Passing Props from Parent to Child
In the parent component, props are passed as attributes.

import React from "react";
import ChildComponent from "./ChildComponent";

function ParentComponent() {
  return <ChildComponent name="John" age={25} />;
}

export default ParentComponent;

Default Props in React
Default props allow you to set fallback values for props in case the parent component does not provide them. This ensures that the component does not break due to missing props.

Using Default Props in Functional Components
In functional components, default values can be assigned directly in the parameter list or using defaultProps.

Method 1: Using Default Parameters (Recommended)

function Greeting({ name = "Guest", age = 18 }) {
  return (
    <h2>
      Hello, {name}! You are {age} years old.
    </h2>
  );
}


Using defaultProps in Class Components
In class components, you can set default props using the defaultProps property.

import React from "react";

class Greeting extends React.Component {
  render() {
    return (
      <h2>
        Hello, {this.props.name}! You are {this.props.age} years old.
      </h2>
    );
  }
}

// Setting default props
Greeting.defaultProps = {
  name: "Guest",
  age: 18,
};

export default Greeting;
