UseEffect :-

The useEffect Hook allows you to perform side effects in your components.
Some examples of side effects are: fetching data, directly updating the DOM, and timers.
useEffect accepts two arguments. The second argument is optional.
useEffect(<function>, <dependency>).

Common Uses of useEffect:

1.Fetching data from APIs
2.Setting up subscriptions or timers
3.Manually changing the DOM
4.Listening for window events (resize, scroll, etc.)

Syntax:
	useEffect(() => {
  	// Side effect logic (e.g., fetch, DOM, timer)
  	return () => {
   	 // Cleanup (optional)
  	};
	}, [dependencies]);

Example 1 :
	useEffect(()=>{
		console.log("Use Effect");
		});
	return(
		<h1>Demo of useEffect {console.log('Inside Component'}</h1>	
	);

	let [count,UpdateCount]=useState(0);
	return(
		<h1>Count :{count}</h1>
		<button onClick="()=>{
				UpdateCount(count+1)
				}">Update</button>	
	)	

Example 2 :
	useEffect(()=>{
		console.log("Use Effect");
		},[empty]);
Example 3 :
	useEffect(()=>{
		console.log("Use Effect");
		},[dependency]);


Important Tips:
->Don't put async directly in useEffect; use an inner async function instead.
->Be cautious with dependencies — unnecessary re-renders may happen.
useEffect is reactive, not imperative — it reacts to state/prop change

Example with fakeapistore:

App.js
import { useEffect, useState } from "react";
import Orders from "./Orders";
import './App.css';

function App(){
    let[product,setProduct]=useState([]);
    useEffect(()=>{
            async function getProducts() {
                let res= await fetch('https://fakestoreapi.com/products');
                let update=await res.json();
                setProduct(update);
                console.log(update);
            }
            getProducts();
        },[])

        if(product.length===0){
            return (
                <h1>Fetching The Data.....</h1>
            )
        }
    return(
        <>  
            <div className="product-list">
            {
                product.map((p)=>
                    <Orders {...p} key={p.id}/>
                )
            }
            </div>
            
        </>
    )
}

export default App;

Orders.js

function Orders({image,title,price,rating}){
    return(
        <>
            <div className="card">
                <img src={image} alt="product_image"/>
                <h4>{title}</h4>
                <p>Price : ${price}</p>
                <p>Rating : {rating.rate}</p>
            </div>
        </>
    )
}

export default Orders;
