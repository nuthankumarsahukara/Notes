In Java, I/O streams are used to read and write data to various sources like files, memory, or network connections. Java I/O (Input/Output) is divided into two main categories:

Byte Streams: Used for handling binary data (e.g., images, audio files).
Character Streams: Used for handling textual data (e.g., files with text).

Types of Streams :

Byte Streams:
Deal with raw binary data.
They read and write data byte by byte.
Commonly used for non-text files (images, audio, etc.).

Classes:
InputStream: Parent class for all byte input streams.
OutputStream: Parent class for all byte output streams.
Subclasses: FileInputStream, FileOutputStream, BufferedInputStream, BufferedOutputStream, etc.

Character Streams:
Deal with characters (text) encoded in a specific character set (e.g., UTF-8).
They read and write data in characters, and are mostly used for text-based I/O operations.

Classes:
Reader: Parent class for all character input streams.
Writer: Parent class for all character output streams.
Subclasses: FileReader, FileWriter, BufferedReader, BufferedWriter, etc.

Java's java.io package contains several classes that allow file operations. These classes can be categorized into Byte Streams (used for binary data) and Character Streams (used for text data).

Basic File Handling Operations in Java
The most common file handling operations include:

1.Creating a file
2.Reading from a file
3.Writing to a file
4.Deleting a file
5.Copying the file

3.Using the filereader class to read a file definding file object

5.Use a bufferread object to read the data using the file reader (or) use a scanner class to read data using file reader.

import java.io.*;
class FileReaderEx
{
	public static void main(String args[])
	{
		String filepath="E:/nuthan/demo123.html";
		File fo=new File(filepath);
		try{
		FileReader fr = new FileReader(fo);
            	BufferedReader buffer = new BufferedReader(fr);
		String data=buffer.readLine();
		while(data != null){
			System.out.println(data);
			data=buffer.readLine();
		}
		}catch(IOException e){
			System.out.println("Unable to send request");
		}
	}
}

Coping the file:

import java.io.*;
class FileCopy
{
	public static void main(String args[])
	{
		InputStreamReader isr=new InputStreamReader(System.in);
		BufferedReader buffer=new BufferedReader(isr);
		FileInputStream fis=null;
		FileOutputStream fos=null;
		try{
			System.out.println("Enter the Source file path:");
			String src=buffer.readLine();
			System.out.println("Enter the Destination file path:");
			String tgt=buffer.readLine();
			File source=new File(src);
			File target=new File(tgt);
			fis =new FileInputStream(source);	
			fos=new FileOutputStream(target);
			//byte array
			byte data[]=new byte[512];
			while(fis.read(data)!=-1){
				fos.write(data);
			}
			fis.close();
			fos.close();
			System.out.println("Data has been copied Successfully");
		}catch(IOException e){
			System.out.println("Unable to read data.....");
		}
	}
}
